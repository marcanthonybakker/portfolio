Midocean implementations

Situation

Midocean, a B2B company that supplies personalized products, wanted to better explain the "Save design as concept" feature and demonstrate the difference between ordering directly and saving as a concept. I also had to investigate IBM's existing Carbon design system because it wasn't working optimally for designers and I wanted to see how it could be improved.

Process

I designed improvements for the feature, created a mobile hamburger menu, and developed a user test plan. I led a user test for the "Communication History" page and recorded the minutes. I also created an alternative design system in Figma, called Pacific, to directly apply my insights from the research.

Result

The project resulted in a clearer user experience, improved functionality, and gave me valuable experience in design systems, user testing, and communicating with a B2B client. This was a really fun client to work for. Often, my ideas weren't exactly what they wanted, but with each iteration, I got closer.
